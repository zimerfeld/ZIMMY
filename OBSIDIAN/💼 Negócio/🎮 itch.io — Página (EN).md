## 🇺🇸 Description (EN — paste as-is)

**Zimmy Pet** is a free, open-source **desktop pet** for Windows. A little procedural
creature lives on your screen — it breathes, blinks, hops, follows your cursor and reacts
when you poke it. Everything is **drawn in code** (no sprites), so every pet is unique.

But Zimmy is more than cute — it's a tiny **assistant**:

- ⏱️ **Visual scheduler** — turn any task into a recurring one (every N minutes, hourly,
  `daily@08:00`…) with a single checkbox.
- 💱 **Currency quotes** — USD, EUR, GBP, JPY, CNY spoken by the pet, flags in the menu.
- 🌤️ **Weather** — current conditions via Open-Meteo (free, no API key), auto-locating you by IP.
- 📧 **Gmail** — unread counter + a soft chime when new mail arrives.
- ⏰ **Alarm & reminders** — Pomodoro, top-of-the-hour cheer, and more.
- 🧩 **Extensible** — drop a `.gd` script in the `Automacoes/` folder and it shows up in the menu.
- 🌐 **Bilingual** — full English / Portuguese (BR) interface, switchable on the fly.

**How to run:** download, unzip if needed, and run `ZimmyPet.exe`. No install, no account.

Free and open source (MIT-style). If Zimmy brightens your desktop, a ⭐ on GitHub or a small
donation keeps it growing. 💜

🔗 Source & issues: https://github.com/zimerfeld/ZIMMY

## 🇧🇷 Description (PT — paste as-is)

O **Zimmy Pet** é um **bichinho de desktop** gratuito e open source para Windows. Uma
criaturinha procedural mora na sua tela — ela respira, pisca, pula, segue o cursor e reage
quando você cutuca. Tudo é **desenhado em código** (sem sprites), então cada pet é único.

Mas o Zimmy é mais que fofura — é um pequeno **assistente**:

- ⏱️ **Agendador visual** — transforme qualquer tarefa em recorrente (a cada N minutos, de
  hora em hora, `daily@08:00`…) com um clique.
- 💱 **Cotações** — USD, EUR, GBP, JPY, CNY faladas pelo bichinho, com bandeirinha no menu.
- 🌤️ **Clima** — tempo atual via Open-Meteo (grátis, sem chave), detectando sua localização por IP.
- 📧 **Gmail** — contador de não lidos + um som suave quando chega e-mail novo.
- ⏰ **Alarme e lembretes** — Pomodoro, comemoração de hora cheia e mais.
- 🧩 **Extensível** — jogue um script `.gd` na pasta `Automacoes/` e ele aparece no menu.
- 🌐 **Bilíngue** — interface completa em Português (BR) / Inglês, troca na hora.

**Como usar:** baixe, descompacte se necessário e rode `ZimmyPet.exe`. Sem instalação, sem conta.

Grátis e open source. Se o Zimmy alegrar sua área de trabalho, uma ⭐ no GitHub ou uma
doação ajudam a manter o projeto crescendo. 💜

🔗 Código & issues: https://github.com/zimerfeld/ZIMMY
