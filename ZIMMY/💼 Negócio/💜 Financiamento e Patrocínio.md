# 💜 Financiamento e Patrocínio

> 🇪🇸 Lee esta página en español → [[💜 Financiamento e Patrocínio (ES)]]

Como o Zimmy Pet é **grátis e open source** (sem venda direta), a monetização é
**indireta**: doações/patrocínio e valor de portfólio. Estratégia de crescimento em
[[🚀 Distribuição e Crescimento]]; métricas em [[📈 Adoção e Métricas]].

## 💰 Canais de apoio
- **GitHub Sponsors:** `zimerfeld` — botão **Sponsor** no repositório
  https://github.com/zimerfeld/ZIMMY.
- **Ko-fi:** `C0D621FCGD` — link de doação avulsa (https://ko-fi.com/C0D621FCGD).
- **Estrela ⭐:** o CTA mais barato e de maior alcance — pedir uma ⭐ no repo em toda copy
  de divulgação ([[📣 Divulgação — Posts]]).

## 🧩 Onde o pedido aparece
- **README** (raiz + PT/EN): seção de apoio com os links de Sponsor/Ko-fi.
- **itch.io:** *Pricing = No payments* com botão **"Donate"** apontando para GitHub
  Sponsors / Ko-fi ([[🎮 itch.io — Página]]).
- **Posts de divulgação:** CTA fixo "⭐ no repositório + 💜 Sponsor (GitHub Sponsors
  `zimerfeld` / Ko-fi `C0D621FCGD`)" — ver [[📣 Divulgação — Posts]].

## 🏷️ Badges (prova social)
No topo do README, badges dinâmicos de **stars** e **adoção** reforçam a tração antes de
pedir apoio:
- ⭐ **Stars:** `https://img.shields.io/github/stars/zimerfeld/ZIMMY`
- 📈 **Adoção/clones:** a métrica viva do projeto é **clones** (janela de 14 dias); os repos
  não publicam assets de release, então *downloads = 0*. Ver [[📈 Adoção e Métricas]] para
  o procedimento de contagem (`gh api .../traffic/clones`).

## 📈 Tese de investimento
O apoio financeiro direto tende a ser modesto num pet grátis; o **retorno real** é
**portfólio + topo de funil** (o pet fica na tela o dia todo, é vitrine dos outros produtos
Zimerfeld) e **prova social** (stars/clones) que sustenta a marca. Baixo custo marginal
(binário único, sem servidor) mantém o risco financeiro perto de zero.
